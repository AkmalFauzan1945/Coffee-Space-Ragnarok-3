Tentang CoffeeSpace

CoffeeSpace dibuat sebagai project Ujian Akhir Semester mata kuliah Web Design. Web ini bertujuan untuk melatih pemahaman dasar HTML, CSS, JavaScript, struktur web, dan desain antarmuka tanpa menggunakan database.

Konsep Project

Website ini menampilkan menu minuman café dan simulasi pemesanan sederhana dengan perhitungan otomatis menggunakan JavaScript.
